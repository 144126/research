## Research Topic: Visual design and UX approaches for BEEE TEAMUP youth chess education program homepage

## Integrated Findings

### Phase 1: SCOPE

**Core Components to Investigate:**
1. Hero section concepts combining chess/education branding with 3D/interactive elements
2. Novel approaches to showing TEAMUP passport/badge system on homepage
3. GSAP/ScrollTrigger storytelling patterns for multi-pillar program explanation
4. Gamification UI patterns (achievement badges, progress rings, XP counters) adapted for website
5. Interactive chess visualization (3D board, animated pieces, move visualization) as hero element
6. Scrollytelling homepage architecture for 6-pillar program with badge system and passport
7. Micro-interaction patterns (custom cursor, magnetic buttons, staggered reveals, scroll counters)
8. Performance strategies for animation-heavy education/homepage sites

**Stakeholder Perspectives:**
- Parents (30-50): Trust, warmth, evidence of educational value
- Children (8-16): Excitement, fun, sense of adventure
- Developer: Practical implementation guidance for SvelteKit + Tailwind v4

**Design System Constraints:**
- Warm cream canvas (#faf9f5), coral/orange primary (#F27830)
- Cormorant Garamond serif display + Inter sans-serif body
- Dark navy product surfaces (#181715) for contrast rhythm
- Rounded corners: 8px buttons, 12px cards, 16px hero
- Section spacing of 96px

**Scope Boundaries:**
- IN: Visual design, UX patterns, animation techniques, micro-interactions, interactive elements
- OUT: Backend implementation, registration form UX, SEO, social media, mobile app design
