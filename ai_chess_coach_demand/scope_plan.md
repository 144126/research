# Phase 1: SCOPE — Research Framing

## Research Question

What is the single most in-demand, underserved, highest-potential software chess tool that would generate massive immediate demand, willingness to pay, and viral adoption upon launch in 2026?

## Core Components Decomposed

1. **Demand identification**: What chess tools do players actively seek but can't find?
2. **Underserved market**: Where is the gap between what exists and what's wanted?
3. **Highest-potential**: Which gap has the largest TAM, willingness-to-pay, and viral mechanics?
4. **Technical feasibility**: Can a solo/small dev team (Svelte 5, Workers, Stockfish) build it?

## Stakeholder Perspectives

- **Club players (1200-1800 ELO)**: Largest segment, want to improve, frustrated with existing tools
- **Advanced players (1800-2200)**: Serious improvers, willing to pay, need depth
- **Beginners (<1200)**: Mass market, need onboarding, low willingness to pay
- **Chess parents**: Buy for kids, value safety and progress tracking
- **Chess coaches**: Want tools to scale their teaching
- **Platform operators**: Chess.com, Lichess — incumbents

## Scope Boundaries

### In Scope
- Chess training/improvement software (all categories)
- AI/LLM-powered chess tools
- Opening preparation tools
- Game analysis platforms
- Puzzle/tactics training systems
- Coaching/mentoring platforms
- Natural language chess analysis
- Pattern recognition tools
- Cross-platform game analysis

### Out of Scope
- Physical chess equipment
- Tournament organization software
- Chess streaming/video platforms
- Enterprise/corporate chess solutions
- Blockchain/crypto chess applications
- Chess for AI/ML research

## Success Criteria
1. Identify ONE clear product opportunity with evidence
2. Quantify market size, WTP, and viral potential
3. Technical feasibility assessment for the dev's stack
4. 270+ sources with citation tracking
5. Data-driven, contrarian-aware recommendations

## Key Assumptions to Validate
1. Chess players WILL pay for the right tool [TEST]
2. Lichess is NOT "good enough" for serious improvers [TEST]
3. LLMs can meaningfully improve chess coaching [TEST]
4. Personalized coaching has more value than generic analysis [TEST]
5. Network effects exist in chess training tools [TEST]
