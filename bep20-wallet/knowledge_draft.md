## Research Topic: Best BEP-20 Wallet

## Integrated Findings

### Software Wallets
- **Trust Wallet**: 220M+ downloads, 100+ chains native, BEP-20 native. Binance-owned. 4.7 avg rating. Security Scanner blocked $191M in 2025. No hardware pairing. 0.7% swap fee. Christmas 2025 hack ($7M, extension v2.68). 53% 1-star reviews from 420 analyzed (support, security anxiety, fees).
- **MetaMask**: 30M+ MAU, EVM-native, deepest dApp ecosystem. 0.875% swap fee. Manual BSC config needed. Native Ledger/Trezor pairing. Best for DeFi power users.
- **Gem Wallet**: 500K+ users, fully open-source, CertiK audited Apr 2026 (11 findings), 4.9 iOS / 4.8 Google Play. Native BEP-20, cross-chain via THORChain. No KYC, no registration. 0 fees (only network gas).
- **Best Wallet**: 60+ chains, no KYC. Built-in DEX aggregator (Rubic). Cloud backup. Tested 3.8/5 vs Trust Wallet 4.5/5. Balance display bugs.
- **Binance Web3 Wallet**: MPC-based, no seed phrase. 9.2/10 Coin Bureau. Integrated in Binance app. Full BEP-20 + staking.
- **Bitget Wallet (BitKeep)**: 100+ chains, $300M protection fund. Trustpilot 2.8/5 (support delays 7-14 days). Formerly BitKeep.
- **OKX Wallet**: 130+ chains native, hardware-backed key storage.
- **Exodus**: 50+ networks, 260+ assets. Best UI design. Swap from 0.5%.
- **Zengo**: MPC seedless, $0 Free / $19.99 Pro. 8.4/10 Coin Bureau. Biometric login.
- **bonuz**: MPC, no seed phrase, Hacken 10/10. Social login (email/Google/Apple).
- **Tether Wallet**: New Apr 2026. @tether.me usernames. No swaps/DeFi/fiat on-ramp.
- **Rabby**: EVM only, 0.25% swap fee. Transaction simulation. Best for advanced EVM users.
- **SafePal app+hardware**: Hybrid. Air-gapped QR signing. $49.99 S1.

### Hardware Wallets
- **Ledger**: 6M+ users. Nano S Plus ($79) EAL6+, Nano X ($149) EAL5+ Bluetooth, Stax ($399) curved e-ink, Flex ($249). Ledger Live. Staking via dApp only. No native BNB staking UI. Ledger Recover controversy (2023). 2025 Recovery Key physical card.
- **Trezor**: Safe 3 ($79) EAL6+, Safe 5 ($169) touch+haptic, Safe 7 ($249) TROPIC01 auditable SE. Fully open-source. Shamir Backup. No iOS app. USB-C only (except Safe 5 BT).
- **SafePal**: S1 ($49.99) air-gapped QR, S1 Pro ($89.99) EAL6+, X1 ($69.99) BT not air-gapped. Binance-backed. 100+ chains. Shorter track record (2018). Build quality concerns.
- **Tangem**: Card form factor, NFC tap. 2-card $54.90 / 3-card $69.90. EAL6+ Samsung SE. No screen = blind signing risk. 25-year warranty. 6M cards shipped.
- **OneKey**: Pro ($278) 91/100, Classic 1S. Fully open-source. SignGuard dual parsing. WalletScrutiny 10/10.
- **ELLIPAL Titan 2.0**: Air-gapped QR, EAL5+, $169. Rugged metal.
- **Keystone Pro 3**: Open-source, $149, 81/100. Air-gapped QR.
- **Cypherock X1**: Shamir Secret Sharing across 5 cards. Unique key-loss protection.

- Bexo Wallet May 2026: $500K drained, 297 wallets

### User Sentiment
- Trust Wallet: 4.7 avg but 53% 1-star reviews on analyzed sample. Main complaints: no support (31%), security anxiety (23%), missing features (19%), hidden fees (15%), bugs (10%)
- Gem Wallet: 4.9 iOS, 4.8 Google Play. Users praise simplicity, speed, open-source
- Bitget Wallet: Trustpilot 2.8/5. Support delays (7-14 days), fund recovery friction
- Best Wallet: 3.8/5 in hands-on testing. Balance display bugs, swap costs 2.4% higher
- MetaMask: 30M+ MAU. Complaints: drain after approvals, high swap fees, gas confusion
- Hardware wallets: Trezor Safe 7 leads for transparency; Ledger for ecosystem; SafePal for budget
