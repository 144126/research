# Phase 2: PLAN - Research Strategy

## Investigation Paths
1. Custom silicon APIs (Cerebras, Groq, SambaNova) - fastest tier
2. GPU-based inference APIs (Together, Fireworks, DeepInfra) - middle tier  
3. Frontier proprietary APIs (OpenAI, Google, Anthropic, xAI) - quality tier
4. TTFT vs throughput analysis
5. Cost-speed tradeoffs
6. Architectural deep dive
7. Criticisms and contrarian views
8. Practical recommendations

## Search Strategy
- Batch 1: Core speed benchmarks and provider comparisons
- Batch 2: Deep dives into individual providers
- Batch 3: Architectural and technical analysis
- Batch 4: Contrarian and critical perspectives
- Batch 5: Use cases and practical implications
- Batch 6: Academic and research sources
- Batch 7: Market analysis and industry reports
- Batch 8: Future trajectories and predictions

## Quality Gates
- 270+ sources minimum
- 3+ independent sources per major claim
- All factual claims cited [N]
- Prose >=80%
- Complete bibliography
- ELI5 inline explanations
