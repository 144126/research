# Research Topic: How to Create Outstanding, Breathtaking, Awwwards-Winning Websites

## Integrated Findings

*This knowledge draft evolves as each source is processed. It accumulates integrated insights from all processed sources, representing the "denoised" state of our research.*

## Key Findings

### 1. Technology Stacks
- GSAP (GreenSock Animation Platform) with ScrollTrigger is the dominant animation library, used in ~40% of Awwwards-winning sites
- WebGL/Three.js is used in ~25% of winners, particularly for immersive experiences
- React/Next.js framework is increasingly common, combined with GSAP or Framer Motion
- Locomotive Scroll and Lenis are the leading smooth-scrolling libraries
- Barba.js, Highway.js, and native View Transitions API for page transitions
- Common CSS features: container queries, cascade layers, view transitions, scroll-driven animations

### 2. Top Agencies
- Locomotive (Canada/France), Makemepulse (France), Active Theory (US), MILL3 (Canada)
- Unseen Studio, Immersive Garden, Merci Michel, MONOGRID, Studio Freight
- French and Canadian agencies dominate the SOTD winners disproportionately

### 3. Design Patterns
- Full-screen sections with sequential reveals
- Scrollytelling/narrative-driven layouts
- Horizontal scroll sections pinned by vertical scroll
- Custom cursor replacements
- Morphing SVG/WebGL transitions between sections
- Particle systems and 3D environments as backgrounds
- Kinetic typography with SplitText

### 4. Performance Considerations
- Most award sites trade some performance for visual impact
- Typical strategies: code splitting, lazy loading, will-change hints, GPU compositing
- INP (Interaction to Next Paint) is the hardest metric for animated sites

### 5. New Technologies (2024-2026)
- View Transitions API for SPA and MPA page transitions
- CSS scroll-driven animations (`scroll-timeline`, `view-timeline`)
- WebGPU for advanced 3D rendering
- OKLCH color space for better color manipulation
- Container queries for responsive component design
