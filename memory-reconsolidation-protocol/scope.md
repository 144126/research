# Phase 1: SCOPE - Research Framing

## Core Question
What is the complete, step-by-step practical protocol for applying memory reconsolidation to change one's own subconscious beliefs?

## Decomposed Components
1. RECON protocol (Armstrong): Recall, Explore, Create mismatch, Objectively describe, Neutralize
2. TRP (Ecker et al.): ABC-123-V framework
3. LessWrong practical guides for self-applied reconsolidation
4. Coherence Therapy methods
5. EMDR mechanisms adapted for self-use
6. Schema access points (Felt Sense, Belief, Evidence, Metaphor)
7. Troubleshooting and safety

## Stakeholder Perspectives
- Self-directed individual (primary)
- Clinical therapist adapting for client self-use
- Rationalist community (LessWrong)

## In Scope
- Step-by-step protocol synthesis
- Practical timing, duration, verification
- Self-application techniques
- Troubleshooting resistance

## Out of Scope
- Pharmacological reconsolidation (propranolol)
- Therapist-only techniques
- Background theory already in Finding 2

## Success Criteria
- Actionable protocol with concrete steps
- 20+ clinical and practical sources
- Verification methods for belief change
- Troubleshooting guidance

## Key Assumptions
- Reconsolidation is the native change mechanism
- Self-application feasible with proper guidance
- Protocols share common core mechanisms
