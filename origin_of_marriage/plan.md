## Triangulation Plan
- Core claims require 3+ independent sources
- Cross-reference: biological (evolutionary data), archaeological (material evidence), ethnographic (cross-cultural records), historical (written records)
- Use phylogenetic studies as high-weight evidence (quantitative)
- Flag religious sources as potentially biased for pre-religious claims

## Quality Gates
- Gate 1: 270+ sources
- Gate 3: All factual claims cited
- Gate 4: ELI5 inline explanations for all technical terms
- Gate 5: 4+ source types including personal/user-reported
- Gate 6: No placeholder citations
- Gate 7: Prose >= 80%
- Gate 8: No "research shows" without citation
- Gate 9: Speculation labeled [SPECULATION]
- Gate 10: Executive summary >= 800 words with citations
- Gate 11: Evidence class labels on every claim
- Gate 12: Weakest evidence self-audit
