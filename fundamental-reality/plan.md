### Knowledge Dependencies
1. Understand QM interpretations → then QG approaches → then metaphysics of QG
2. Understand hard problem → then IIT/panpsychism → then combination problem
3. Understand information theory → then digital physics → then constructor theory
4. Understand time in physics → then problem of time → then cosmological natural selection

### Triangulation Approach
- Every major claim: 3+ independent sources from different traditions
- Physics claims: cross-reference academic + encyclopedia + popular science
- Controversial claims: seek both proponent and critic sources
- Survey data: use as ground truth for consensus/disagreement
